# ফোটনকার্ড—ল্যান্ডিং পেজ (বাংলা)

এটি **FotonCard** ভার্চুয়াল ক্রেডিট কার্ড সেবার জন্য একটি স্ট্যাটিক ওয়েব পেজ। বিষয়বস্তু **বাংলা ভাষায়** লেখা, যাতে বাংলাদেশ ও বাংলাভাষী ব্যবহারকারীরা সহজে পড়তে পারেন।

## এই প্রজেক্টে কী আছে

| ফাইল | বর্ণনা |
|------|--------|
| `index.html` | মূল HTML পেজ—মেটা ট্যাগ, স্টাইল, FAQ ও রেজিস্ট্রেশন লিংক |
| `favicon.ico` | সাইট আইকন |
| `*.jpg` | স্ক্রিনশট ও চিত্র (চার্জ, সাইনআপ, BIN, লেনদেন ইত্যাদি) |

## স্থানীয়ভাবে দেখার উপায়

1. `index.html` ফাইলটি ব্রাউজারে খুলুন (ডাবল ক্লিক বা “Open with” দিয়ে)।
2. অথবা সহজ HTTP সার্ভার চালু করুন (ঐচ্ছিক):

```bash
# Python 3
python -m http.server 8080
```

তারপর ব্রাউজারে `http://localhost:8080` খুলুন।
